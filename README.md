# Calculadora Web

Uma calculadora moderna desenvolvida com **HTML, CSS e JavaScript**, inspirada nas calculadoras de smartphones.
O projeto possui interface elegante, responsiva e animações que tornam a experiência de uso mais agradável.

# Demonstração

Interface inspirada em calculadoras mobile, com design escuro e destaque em dourado.

# Tecnologias utilizadas

* **HTML5** – Estrutura da aplicação
* **CSS3** – Estilização e layout responsivo
* **JavaScript** – Lógica das operações matemáticas

# Funcionalidades

* Operações matemáticas básicas:

  * ➕ Soma
  * ➖ Subtração
  * ✖ Multiplicação
  * ➗ Divisão
* Porcentagem (%)
* Alternar sinal (+/−)
* Número decimal
* Apagar último número (⌫)
* Limpar cálculo (C)
* Cálculos encadeados
* Interface moderna com animações nos botões
